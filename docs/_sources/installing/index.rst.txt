Installing scProximitE
======================

Python using pip
----------------
.. code-block:: python

    pip install scproximite



Note: scproximite was developed using Python 3.8, if you have any issues we recommend using conda and creating a new
environment before installing:


.. code-block:: bash
    conda create --name scproximite python=3.8
    conda activate scproximite
    pip install scproximite
