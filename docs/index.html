scProximitE

scProximitE is a framework for the in-depth evaluation of 17 proximity metrics performance with respect to core structural properties of scRNA-seq data, including sparsity, dimensionality, cell population distribution and rarity.

Running scProximitE

Check out our install instructions and then try out the tutorials.

Extending scProximitE

If you want to add your new distance metrics that’s great, feel free to add any new ones!

  1. Make a pull request on github - we have made the code extendable for the loss function etc.

Citing scProximitE

Sci-viso can be cited as in References, where we also provide citations for the used tools.
